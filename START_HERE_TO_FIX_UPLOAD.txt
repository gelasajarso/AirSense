┌────────────────────────────────────────────────────────────────────────────┐
│  STEP 1: STOP THE API SERVER                                              │
└────────────────────────────────────────────────────────────────────────────┘

  Find the terminal window that shows:
  
      INFO:     Uvicorn running on http://0.0.0.0:8000
  
  Press:  Ctrl + C
  
  ✓ You should see "Shutting down" message


┌────────────────────────────────────────────────────────────────────────────┐
│  STEP 2: START THE API SERVER AGAIN                                       │
└────────────────────────────────────────────────────────────────────────────┘

  In the same terminal, type:
  
      python -m uvicorn src.api.main:app --host 0.0.0.0 --port 8000 --reload
  
  Press: Enter
  
  ✓ Wait for "Application startup complete" message


┌────────────────────────────────────────────────────────────────────────────┐
│  STEP 3: VERIFY IT WORKS                                                  │
└────────────────────────────────────────────────────────────────────────────┘

  Open your browser and go to:
  
      http://localhost:8000/docs
  
  ✓ Look for "POST /upload" in the list
  
  If you see it, you're done! Try uploading again.

  ❓ Can't find the API terminal?
  
     1. Open Task Manager (Ctrl+Shift+Esc)
     2. Find "Python" processes
     3. End the one using port 8000
     4. Start API server again
  
  
  ❓ Still getting 404 error?
  
     1. Make sure you restarted the API (not the dashboard)
     2. Check http://localhost:8000/docs shows /upload
     3. Try uploading again
  
  
  ❓ Port 8000 already in use?
  
     Run: netstat -ano | findstr :8000
     Kill that process or use a different port
